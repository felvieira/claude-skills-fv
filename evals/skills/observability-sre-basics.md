# Eval - Observability SRE Basics

## Objetivo
Validar que a skill cobre logs, metricas, tracing, health checks e rollback sem inflar a solucao.

## Entrada
Servico web com deploy existente e pouca observabilidade.

## Esperado
- checklist base aplicado
- riscos operacionais priorizados
- handoff claro para Backend, Deploy ou Documenter

## Evidencias Minimas
- sinais criticos definidos
- lacunas registradas
- recomendacoes acionaveis

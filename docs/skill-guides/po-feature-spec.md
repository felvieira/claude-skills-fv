# PO Feature Spec Guide

Guia auxiliar da skill `01-po-feature-spec` para exemplos completos de spec, priorizacao e criterios de aceitacao.

## Conteudo sugerido desta skill
- template completo de feature spec
- matriz de priorizacao
- exemplos bons e ruins de criterio de aceitacao
- checklists de aprovacao entre areas

## Uso
- consultar apenas quando a tarefa exigir spec detalhada ou alinhamento formal entre areas
- preferir a estrutura curta da skill para requests simples

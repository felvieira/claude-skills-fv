# Repo Auditor Guide

Guia auxiliar da skill `18-repo-auditor` para auditorias mais completas e revisoes incrementais.

## Conteudo sugerido desta skill
- checklist de stack e tooling
- deteccao de convencoes de codigo e docs
- mapeamento de auth, testes, deploy e observabilidade
- leitura de assets e identidade visual existente
- estrategia de reauditoria incremental

## Uso
- consultar apenas quando a auditoria exigir aprofundamento maior
- preferir `docs/repo-audit/current.md` para uso cotidiano e economia de token

# Frontend Integration Guide

Guia auxiliar da skill `04-frontend-integration` para exemplos extensos, estruturas de pasta e snippets de referencia.

## Conteudo sugerido desta skill
- estrutura de pastas por tipo de app
- exemplos de store, api client e refresh flow
- padroes de loading, erro, vazio e stale state
- exemplos de hooks e integracao de formularios

## Uso
- consultar apenas quando a tarefa exigir exemplo concreto
- nao carregar este guia por padrao em tasks simples

# Orchestrator Playbook

Guia auxiliar da skill `09-orchestrator` para pipelines extensos, cenarios reais e fluxos de rejeicao.

## Conteudo sugerido desta skill
- pipelines por tipo de task
- regras de re-execucao e rollback de etapa
- exemplos de plano de execucao
- exemplos de classificacao de risco e dependencia

## Uso
- consultar apenas quando o pipeline fugir do caminho padrao
- preferir o core curto da skill para classificacao inicial

# Documenter Templates Guide

Guia auxiliar da skill `10-documenter` para templates completos de feature, ADR, runbook e playbook.

## Conteudo sugerido desta skill
- template de feature doc
- template de ADR
- template de runbook
- template de playbook de incidente

## Uso
- consultar apenas quando precisar gerar docs novas ou padronizar artefatos
- preferir `templates/` para saidas curtas do fluxo corrente

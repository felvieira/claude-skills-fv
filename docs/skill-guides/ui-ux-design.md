# UI UX Design Guide

Guia auxiliar da skill `02-ui-ux-design` para tokens, exemplos de layout, heuristicas e acessibilidade.

## Conteudo sugerido desta skill
- tokens e design system base
- wireframes e fluxos de navegacao

- heuristicas de usabilidade e estados de interface

## Uso
- consultar apenas quando a tarefa exigir exemplos detalhados de interface
- preferir a skill principal para direcao curta e handoff

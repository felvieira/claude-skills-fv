# Observability SRE Guide

Guia auxiliar da skill `19-observability-sre` para playbooks, sinais e recomendacoes operacionais detalhadas.

## Conteudo sugerido desta skill
- logs estruturados e redacao de dados sensiveis
- metricas essenciais por tipo de servico
- tracing e correlacao entre camadas
- health checks, readiness, rollback e alertas
- runbooks de incidente e operacao

## Uso
- consultar apenas quando a tarefa exigir detalhamento operacional maior
- preferir a skill principal para checklist curto e handoff

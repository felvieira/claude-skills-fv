# QA Testing Guide

Guia auxiliar da skill `05-qa-testing` para configuracoes extensas, fixtures e exemplos de testes.

## Conteudo sugerido desta skill
- setup de Vitest e Playwright
- exemplos de mocks com MSW
- fixtures autenticadas
- smoke tests, regression checklist e baseline de performance

## Uso
- consultar apenas quando houver implementacao ou ajuste real de estrategia de testes
- evitar carregar este guia inteiro para requests simples de review

# Motion Design Guide

Guia auxiliar da skill `12-motion-design` para tokens, variants e exemplos de transicao.

## Conteudo sugerido desta skill
- motion tokens
- padroes de entrada, saida e stagger
- acessibilidade com reduced motion
- performance e limites de animacao

## Uso
- consultar apenas quando a tarefa exigir implementacao ou padrao detalhado de motion
- preferir a skill principal para direcao curta

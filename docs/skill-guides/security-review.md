# Security Review Guide

Guia auxiliar da skill `06-security-review` para checklists detalhados, exemplos de findings e mitigacoes.

## Conteudo sugerido desta skill
- checklist OWASP detalhado
- exemplos de findings por severidade
- requisitos de auth, dados sensiveis e headers
- dependencia, audit e review de configuracao

## Uso
- consultar apenas quando a revisao exigir profundidade maior
- preferir a skill principal para classificacao de risco e handoff
